# GO-DAL - GO Database Abstraction Layer

GO-DAL is a lightweight, interface-driven database abstraction layer for Go that allows developers to write database-agnostic SQL queries (Insert, Select, Update, Delete) across major databases like MySQL, PostgreSQL, and SQLite. The library provides a simple API for building SQL queries and executing them against different database backends without worrying about the underlying syntax differences.

## Features

- Support for MySQL, PostgreSQL, and SQLite databases
- Interface-driven design for easy extensibility
- Query builder for constructing SQL queries programmatically
- Database driver wrapper for executing queries and scanning results
- Unit tests to ensure reliability and correctness
